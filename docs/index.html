Rato no Labirinto

Este projeto foi desenvolvido como parte do Projeto Integrador de Engenharia. A proposta é construir um sistema funcional em que um rato percorre um labirinto, tomando decisões até encontrar a saída de forma eficiente.

Projeto publicado no GitHub Pages

Aqui apresentamos a proposta, os objetivos, o funcionamento do sistema e a evolução do projeto.

O foco é demonstrar uma solução prática, organizada e funcional para o problema do rato no labirinto.

Sobre o projeto

Contexto

O projeto foi criado para aplicar conceitos de engenharia, lógica, programação e resolução de problemas em uma aplicação prática.

Problema

O desafio consiste em fazer um rato encontrar a saída de um labirinto, utilizando regras, sensores, algoritmos ou estratégias definidas pela equipe.

Solução

ainda em desenvolvimento

Objetivos

Objetivo geral

Construir um projeto funcional que simule ou controle o deslocamento de um rato em um labirinto.

Objetivos específicos

Implementar a lógica de movimentação, testar diferentes caminhos e validar o funcionamento do sistema.

Resultados esperados

Demonstrar que o rato consegue navegar pelo labirinto e alcançar a saída com consistência.

Como funciona

Entrada

O sistema recebe a estrutura do labirinto e define a posição inicial do rato.

Processamento

A lógica do projeto analisa os caminhos possíveis e decide os movimentos a serem executados.

Saída

O resultado é a trajetória realizada pelo rato até encontrar a saída do labirinto.
