Orçamento do Projeto Micromouse

Consolidação das opções de componentes avaliadas para o Micromouse, organizadas por categoria, com foco em custo, peso, integração e viabilidade técnica para o projeto.

Resumo do orçamento

Os componentes foram organizados por função no projeto para facilitar a comparação entre alternativas. Sempre que aplicável, as opções foram classificadas como melhor custo-benefício, melhor possível e mínimo viável, permitindo uma leitura rápida para tomada de decisão.

Categorias

2

Energia e componentes eletrônicos concentrando os principais itens de compra do protótipo.

Estrutura da análise

3 níveis

Melhor custo-benefício, melhor possível e mínimo viável quando a comparação foi disponibilizada.

Uso

Apresentação

Tabela limpa e consolidada para discussão de compras, revisão técnica e decisão de integração.
